# Vitis AI - Akıllı Antrenör

**Vitis AI**, yanlış egzersiz formlarını gerçek zamanlı olarak tespit eden ve kullanıcıyı anında uyaran, yapay zeka destekli bir güvenlik ve fitness koçluk sistemidir.

## 🚀 Proje Vizyonu
Yanlış antrenmanla vakti ve sağlığı riske atmayı önlemek amacıyla tasarlanan bu proje, sakatlıkları önleyen bir güvenlik kalkanı gibi çalışır. Teknoloji burada sadece bir performans aracı değil, hayat kurtaran koruyucu bir sistemdir.

## ✨ Özellikler
- **Gerçek Zamanlı Form Analizi:** Saniyede 30 kare hızında anlık iskelet takibi.
- **Biyomekanik Hassasiyet:** Hatalı duruşların matematiksel olarak algılanması.
- **Haptik ve Sesli Uyarılar:** Ekrana bakmadan, akıllı saat (titreşim) ve kulaklık (ses) ile yönlendirme.
- **Acil Durum Protokolü:** Boğulma veya uzun süreli hareketsizlik algılandığında acil durum kişilerine (veya 112'ye) konum gönderimi.
- **Offline (On-Device) Çalışma:** İnternet olmadan da kesintisiz koruma.

## 🛠️ Teknolojiler
- HTML5, Vanilla CSS3 (Custom Properties, Glassmorphism, CSS Grid/Flexbox)
- Vanilla JavaScript (IntersectionObserver, DOM manipülasyonu, IIFE tabanlı simülasyonlar)
- Responsive (Mobil Uyumlu) Tasarım

## 📦 Kurulum ve Çalıştırma
Projeyi yerel bilgisayarınızda çalıştırmak için:
1. Depoyu klonlayın: `git clone https://github.com/emirhan-oner/vitis.ai.git`
2. `index.html` dosyasını tarayıcınızda açın veya VS Code üzerinden "Live Server" eklentisiyle çalıştırın.
